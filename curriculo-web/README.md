# Currículo Web

Projeto de um currículo desenvolvido em **HTML**, com o objetivo de praticar estruturação de conteúdo e apresentação visual.

## Tecnologias utilizadas
- HTML  
- CSS  

## Objetivo
Criar uma versão simples de currículo em formato web, aplicando boas práticas de organização e semântica.

Projeto de estudo e prática em desenvolvimento web.
