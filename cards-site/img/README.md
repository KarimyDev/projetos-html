# Pasta de Imagens

Esta pasta contém todas as imagens utilizadas no projeto do site de doação.

## Tipos de imagens
- Logo do projeto
- Imagem de capa
- Imagens dos cards de doação

## Padrão de nomes
Os arquivos seguem um padrão de nomes em letras minúsculas, separados por hífen, para facilitar a organização e manutenção do projeto.

### Exemplos:
- logo.webp  
- capa.webp  
- card-doacao-1.webp  
- card-doacao-2.webp  

## Observações
As imagens são utilizadas apenas para fins educacionais e de estudo em desenvolvimento web.
